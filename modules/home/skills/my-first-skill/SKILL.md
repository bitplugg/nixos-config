# My First Skill

## Описание
Этот скилл демонстрирует базовую структуру для создания скиллов в OpenClaw.

## Когда использовать
- Для автоматизации повторяющихся задач.
- Для интеграции с внешними API или инструментами.

## Инструкции
1. Поместите скрипты в `scripts/`.
2. Добавьте документацию в `references/`.
3. Используйте `assets/` для медиафайлов.

## Пример использования
```bash
openclaw skills run my-first-skill --task "Пример задачи"
```
